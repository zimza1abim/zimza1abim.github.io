## 인프라 활용을 위한 파이썬 프로그래밍 / 1주차 3일자 수업

이번 시간에는 리스트, 튜플, 딕셔너리에 대해 알아보기로 합니다. 숫자와 문자열만으로는 복잡한 특정 조건을 표현하는게 쉽지가 않습니다. 그렇기 때문에 리스트, 튜플, 딕셔너리는 매우 유용하게 사용할 수 있습니다.
{:.lead}

### 1. 리스트(List)

#### 1-1. 리스트 기본

리스트(List)?
> 1. 리스트를 사용하면 홀수나 짝수와 같은 숫자 모음을 간단하게 표현할 수 있음.<br>
> 2. 리스트 작성시 '대괄호[]'를 사용하며 대괄호 사이에 요소값을 넣음.<br>
> 3. 요소값 사이는 '쉼표,'를 사용하여 구분할 수 있음.<br> 
{:.lead}

- 리스트 기본 예시 📋

```python
리스트명 = [요소1, 요소2, 요소3, ....]
a = [1, 2, ['python', 'java']]
b = [1, 2, 3]
c = ['python', 'java', 'csharp', 'golang']
```

- 리스트 활용 예시 📋

```python
numlist = [0, 0, 0, 0]
sum = 0

numlist[0] = int(input("숫자 : "))
numlist[1] = int(input("숫자 : "))
numlist[2] = int(input("숫자 : "))
numlist[3] = int(input("숫자 : "))

sum = numlist[0] + numlist[1] + numlist[2] + numlist[3]
print("sum -> ", sum)
```

실행 결과 ⬇️<br>
`숫자 : 3`<br>
`숫자 :  2`<br>
`숫자 : 1`<br>
`숫자 : 4`<br>
`sum ->  10`<br>

```python
numList = []
for i in range(0, 4) :
  numList.append(0)
sum = 0

for i in range(0, 4) :
  numList[i] = int(input( " 숫자 : " ))

sum = numList[0] + numList[1] + numList[2] + numList[3]

print("합계 ==> ", sum)
```

실행 결과 ⬇️<br>
`숫자 : 8`<br>
`숫자 : 77`<br>
`숫자 : 888`<br>
`숫자 : 9999`<br>
`합계 ==>  10972`<br>

#### 1-2. 리스트 인덱싱과 슬라이싱

- 리스트의 인덱싱 📋

```python
a = [1, 2, 3]
a[0] -> 1 #a0이 리스트의 첫 번째 요소!!!!!
a[1] -> 2
a[0] + a[2] -> 1 + 3 = 4
```

- 리스트의 슬라이싱 📋

```python
a = [1, 2, 3, 4, 5]
b = a[:2] #처음부터 a[1]까지를 나타냄
c = a[2:] #a[2]부터 끝까지를 나타냄

b -> [1, 2]
c -> [3, 4, 5]
```

#### 1-3. 리스트 연산 수행

- 리스트의 연산(+더하기) 📋

```python
a = [1, 2, 3]
b = [4, 5, 6]
a + b
```

실행 결과 ⬇️<br>
`[1, 2, 3, 4, 5, 6]`<br>

- 리스트의 연산(*반복하기) 📋

```python
a = [1, 2, 3]
a * 3
```

실행 결과 ⬇️<br>
`[1, 2, 3, 1, 2, 3, 1, 2, 3]`<br>

- 리스트 길이 구하기(len) 📋

```python
a = [1, 2, 3]
len(a)
```

실행 결과 ⬇️<br>
`3`<br>

#### 1-4. 리스트 수정, 삭제

```python
a = [1, 2, 3]
a[2] = 4 #리스트에서 값 수정하기
a
[1, 2, 4]
```

`a -> [1, 2, 4]`<br>

```python
a = [1, 2, 3]
del a[1] #리스트에서 값 삭제하기
a
```

#### 1-5. 리스트 관련 func() part1. append, sort, reverse, index

- 리스트 요소 추가 append 📋

```python
a = [1, 2, 3]
a.append(4) #append 함수는 리스트의 맨 마지막에 추가해야 함
a
[1, 2, 3, 4]
```

실행 결과 ⬇️<br>
`a -> [1, 2, 3, 4]`<br>
append 함수는 리스트의 맨 마지막에 추가되는 점.

- 리스트 정렬 sort 📋

```python
a = [1, 4, 3, 2]
a.sort( )
a
[1, 2, 3, 4]
```

실행 결과 ⬇️<br>
`a -> [1, 2, 3, 4]`<br>
sort 함수는 리스트의 값을 순서대로 정렬해줌. 문자와 숫자 둘 다 정렬 가능.

- 리스트 뒤집기 reverse 📋

```python
a = ['a', 'c', 'b']
a.reverse( )
a
['b', 'c', 'a']
```

실행 결과 ⬇️<br>
`a -> ['b', 'c', 'a']`<br>
reverse 함수는 순서대로 역순 정렬이 아니라, 현재 그대로의 값을 역순으로 뒤집는다.

- 리스트 위치 반환 index 📋

```python
a = [1,2,3]
a.index(3)
2
a.index(1)
0
```

실행 결과 ⬇️<br>
`a.index(3) -> 2`<br>
`a.index(1) -> 0`<br>
여기서 헷갈리는 점 주의) (3)은 [1,2,3] 중 a[2]값이다.<br>
마찬가지로 (1)은 a[0]값임. 엄청 헷갈리니 주의해야 함.

#### 1-5. 리스트 관련 func() part2. insert, remove, pop, count

- 리스트 요소 삽입 insert 📋

```python
a = [1, 2, 3]
a.insert(0, 5) #a[0]위치에 5를 삽입한다.
a.insert(3, 6) #a[3]위치에 6을 삽입한다.
```

실행 결과 ⬇️<br>
`[5, 1, 2, 3]`<br>
리스트는 a번째 위치에 b를 삽입하는 함수이다.

- 리스트 요소 제거 remove 📋

```python
a = [1, 2, 3, 4, 5, 6 ,3]
a.remove(3) #첫 번째로 나오는 3을 삭제하라.
[1, 2, 4 ,5 , 6, 3]
```

실행 결과 ⬇️<br>
`[1, 2, 4, 5, 6, 3]`<br>
remove는 첫 번째로 나오는 값만 삭제하는 것. n번째를 삭제하려면 여러번 수행하면 된다.

- 리스트 요소 꺼내기 pop 📋

```python
a = [1, 2, 3]
a.pop() #요소 꺼내기. 만약 a.pop(1)이면 a[1]인 2를 꺼내는 것.
3 #3을 꺼냄
a #a출력
[1,2] #최종 리스트 값
```

실행 결과 ⬇️<br>
`a.pop(3) -> a = [1, 2]`<br>
pop 함수는 리스트의 맨 마지막 요소를 돌려주고 그 요소는 삭제하는 것.

- 리스트의 요소 개수 세기 count 📋

```python
a = [1, 2, 3, 1, 2]
a.count(1)
2
```

실행 결과 ⬇️<br>
`a.count(1) -> 2`<br>
1이라는 값이 리스트에 2개있으ㄹ므로 count값을 2로 출력한다.


### 2. 튜플(tuple)

#### 2-1. 튜플 기본

튜플?
> 1. 튜플은 ()을 사용한다.<br>
> 2. 튜플은 리스트와 거의 유사하지만 **값의 생성/삭제/수정이 불가능**하다.<br>
> 3. 튜플은 리스트와 달리 (1,) 처럼 1개의 요소만을 가질때 콤마(,)를 붙여야 한다.<br>
> 4. 괄호() 없어도 사용가능하다.<br>
{:.lead}

- 튜플 기본 예시 📋

```python
t1 = ( )
t2 = (1,)
t3 = (1, 2, 3)
t4 = ('a', 'b', ('ab', 'cd'))
```

- 튜플 값의 삭제 불가 예시 📋

```python
t1 = (1, 2, 'a', 'b')
del t1[0]
```

실행 결과 ⬇️<br>
`Traceback (most recent call last):`<br>
`File "<stdin>", line 1, in <module>`<br>
`TypeError: 'tuple' object doesn't support item deletion`<br>

- 튜플 인덱싱 예시 📋

```python
t1 = (1, 2, 'a', 'b')
t1[0]
1
t1[3]
'b'
```

실행 결과 ⬇️<br>
`t1[0] -> 1`<br>
`t1[3] -> b`<br>

- 튜플 슬라이싱 예시 📋

```python
t1 = (1, 2, 'a', 'b')
t1[1:]
(2, 'a', 'b')
```

실행 결과 ⬇️<br>
`t1[1:] -> (2, 'a', 'b')`<br>

### 3. 딕셔너리(Dictionary)

#### 3-1. 딕셔너리 기본

딕셔너리?
> 1. 딕셔너리는 연관배열 형식으로 대응관계를 나타내는 자료형 구조이다.<br>
> 2. 딕셔너리는 key와 value 한 쌍을 가지고 있다. ex) 이름 = 홍길동 <br>
> 3. 딕셔너리는 key를 통해 value값을 얻는다. 즉, 모두 찾는게 아니라 key값만 찾는다.<br>
> 4. 쌍 추가, 삭제가 가능하다.<br>
{:.lead}

- 딕셔너리 기본 예시 📋

```python
{Key1:Value1, Key2:Value2, Key3:Value3,....} #기본 구조
dic = {'name':'moon', 'phoneNum':'010-222', 'city': 'seoul'}
```

- 딕셔너리 함수를 이용한 keys 리스트 📋

```python
dic = {'이름':'twice', '구성원 수':'9', '데뷔':'서바이벌 식스틴', '대표곡':'cry for me'}
dic.keys()
for k in dic.keys():
  print(k, "->", dic[k])
```

실행 결과 ⬇️<br>
`이름 -> twice`<br>
`구성원 수 -> 9`<br>
`데뷔 -> 서바이벌 식스틴`<br>
`대표곡 -> cry for me`<br>
